Introduction

Spypal makes it easy to quickly generate attractive colormaps for spyview or other applications where uniform colorscales are handy.

Spypal has two basic modes: waypoint pathfinder, and optimal pathfinder. At present, optimal pathfinder is almost useless.

Each time a new colormap is generated, it is saved as a "cmap.ppm" as an image and "cmap.txt" as human readable text. The colormap can also be saved into a specific file using the "Export" options in the "File" menu.
